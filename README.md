🚀 Mi Portafolio Profesional
Este es el repositorio central de mi portafolio web, desarrollado para mostrar mis proyectos, habilidades técnicas y experiencia como Desarrollador Full Stack.

🌐 Acerca del Proyecto
El sitio está construido utilizando Node.js y Express, siguiendo el patrón de arquitectura MVC (Modelo-Vista-Controlador). Su objetivo es presentar de forma profesional y dinámica mis trabajos más relevantes, integrando tecnologías modernas para la gestión de datos y despliegue en la nube.

🛠️ Stack Tecnológico
Backend: Node.js, Express

Frontend: EJS (Template Engine), HTML5, CSS3, Bootstrap, Chart.js

Base de Datos: Experiencia técnica con MongoDB, MySQL, PostgreSQL, SQLite y TiDB Cloud.

Infraestructura & CI/CD: Vercel (Despliegue continuo)


📂 Estructura del Repositorio
/
├── bin/                # Script de inicio
├── data/               # Archivos JSON/JS con la data de proyectos y experiencia
├── public/             # Archivos estáticos (CSS, imágenes, JS del frontend)
├── routes/             # Controladores de rutas y navegación
├── views/              # Vistas EJS (Layouts, partials, páginas)
├── app.js              # Configuración principal de la aplicación
└── package.json        # Dependencias y scripts

📦 Instalación Local
Para ejecutar este proyecto en tu entorno local:

Clonar el repositorio:

Bash
git clone https://github.com/luigi1501/mi-portafolio
Instalar dependencias:

Bash
npm install
Ejecutar el servidor:

Bash
npm start
